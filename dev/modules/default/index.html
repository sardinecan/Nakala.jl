Default

Le module Nakala.Default correspond au end-point Nakala default. Il permet de vérifier l'état des ressources dans les services ElasticSearch et Datacite.

Nakala.Default.getresourceprocessing — Method
getresourceprocessing(identifier::String, headers::Dict, apitest::Bool=false)

État d'une ressource désignée par identifier dans ElasticSearch et Datacite.
